# 付费会员设计文档

## 黄金会员

**价格**: 5元/月

黄金会员适用于个体用户、小微型企业。相比于普通用户，黄金用户可拥有团队空间、超大储存容量、更高的下载速度与无限制的文档转换次数等诸多权益。

## 超级会员

**价格**: 20元/月

超级会员适用于企业用户。除了享受黄金会员的权益外，超级会员能享受更大的公共空间。
